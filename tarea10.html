IDE para editar y ejecutar Gödel
    
Para compilar en Gödel, debido a que el compilador de Gödel compila el código fuente en el código fuente de Prolog, se usará el mismo IDE con el que trabajamos en Prolog.
SWI-Prolog



SWI-Prolog es una implementación en código abierto (en inglés, open source) del lenguaje de programación Prolog. Su autor principal es Jan Wielemaker. En desarrollo ininterrumpido desde 1987, SWI-Prolog posee un rico conjunto de características, bibliotecas (incluyendo su propia biblioteca para GUI, XPCE), herramientas (incluyendo un IDE) y una documentación extensiva. SWI-Prolog funciona en las plataformas Unix, Windows y Macintosh.
El nombre SWI deriva de Sociaal-Wetenschappelijke Informatica ("Informática de Ciencias Sociales"), el antiguo nombre de un grupo de investigación en la Universidad de Ámsterdam en el que Wielemaker está integrado. El nombre de ese grupo se cambió posteriormente a HCS (Human-Computer Studies).

¿A qué tipo de paradigma pertenece?

El lenguaje de programación Gödel es un lenguaje de programación declarativo, adherido al paradigma de la programación lógica.

Un ejemplo de un programa escrito en ese lenguaje

El siguiente ejemplo es una especificación del máximo común divisor (MCD) de dos números.
MODULE      MCD.
IMPORT      Integers.
 
PREDICATE   Mcd : Integer * Integer * Integer.
Mcd(i,j,d) <- 
           DivisorComun(i,j,d) &
           ~ SOME [e] (DivisorComun(i,j,e) & e > d).
 
PREDICATE   DivisorComun : Integer * Integer * Integer.
DivisorComun(i,j,d) <-
           IF (i = 0 \/ j = 0)
           THEN
             d = Max(Abs(i),Abs(j))
           ELSE
             1 =< d =< Min(Abs(i),Abs(j)) &
             i Mod d = 0 &
             j Mod d = 0.
